## 🌟 Why AEGIS?
AEGIS (Adaptive Engineering Governance & Intelligence System) v5.1 is not just another collection of prompts. It is a **fully-fledged AI Engineering Operating System & Knowledge Orchestration Layer**. 

By combining the **AEGIS Execution Protocol** (how to think & act safely) with **AEOS Domain Knowledge** (what to build), it wraps around base AI models (Claude, Gemini, GPT) like a transmission, driving consistency and avoiding regressions.

## ✨ Core Architecture
AEGIS operates as an autonomous runtime governed by the **Cognitive Scheduler**. The Scheduler manages the execution lifecycle of **24 cognitive engines** through an adaptive loop:

- 🧠 **Intent Analyzer**: Initializes the Global Runtime Context Object.
- ⚙️ **Cognitive Scheduler**: Selects and sequences execution engines based on complexity tiers.
- ⚖️ **Consensus & Judge**: Resolves conflicts between multiple agentic engines.
- 🛡️ **Critic Engine**: Performs self-criticism & edge-case audits.
- 🎯 **Quality Gate**: Calculates overall scores (Target ≥95) before finalizing outputs.

## 🚀 Key Features

### 1. Invariants Defense
Non-negotiable rules enforcing evidence hierarchy, tool honesty, and prompt injection defense. AEGIS will never execute hostile commands hidden in code.

### 2. Multi-Agent Collaborations
- **Feedback Loop**: Introduces the *Observe & Reflect* cycle. If a compilation or test command fails, the agent must inspect the error log and update its strategy before modifying files again.
- **Local Conventions Adoption**: Dynamically reads local .coderules, .cursorrules, or CONVENTIONS.md files at the start of the session to adapt to repository-specific rules.

### 3. Self-Learning Layer
AEGIS learns from its mistakes. The framework extracts patterns from execution failures and updates the local .aegis/patterns/ knowledge base to prevent future identical errors.

### 4. Complexity Tiers Arbitration
Tasks are classified dynamically by blast radius (T0 to T3). High-risk tasks (like Auth, DB schemas, or Payments) automatically require explicit manual verification.
