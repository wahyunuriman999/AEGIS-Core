# Technical Debt

- Track debt explicitly (ticket/TODO with context), don't let it live only in someone's memory.
- Distinguish deliberate debt (a conscious shortcut, documented) from accidental debt (found later, unplanned) — both get reported, only the first was a choice.
- Report debt noticed near your change even if you didn't fix it (Engine 20: Technical Debt Identified) — visibility is the first step to paying it down.
