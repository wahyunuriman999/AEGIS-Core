# Glossary

- **Diff locality** — keeping a change confined to the files actually required by the stated problem.
- **Blast radius** — the set of things that could break if this change is wrong.
- **Root cause** vs **symptom** — the underlying condition that produces a bug, vs. its visible effect.
- **Breaking change** — any change that requires a consumer of this code to also change.
- **Tech debt** — a deliberate or accidental shortcut that will cost more to fix later than now.
